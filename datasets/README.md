# 自建数据集协作规则

训练图片和标签可能快速增长，并可能包含人物、宿舍或实验室环境，因此本目录只保存规则，不把原始数据直接提交到Git。

建议的数据集版本结构：

```text
dataset-v1/
  images/train  images/val  images/test
  labels/train  labels/val  labels/test
  data.yaml
  SOURCES.csv
```

- 类别固定为 `drone` 和 `fire_extinguisher`，合并外部数据时统一类别编号。
- 同一视频连续帧、同一原图的亮度增强版本必须放在同一数据划分，防止训练集泄漏到测试集。
- 每张外部图片在 `SOURCES.csv` 记录来源、许可、原始链接和采集日期。
- 删除重复、严重模糊、错误标注和包含隐私但未获授权的图片。
- 测试集保留真实弱光、逆光、远距离、小目标、遮挡和相似红色物体，不只使用人工调暗图片。
- 用版本号或日期发布压缩包，并记录文件SHA256；组员通过学校网盘或对象存储共享，Git只保存说明和小型统计文件。
